This Common Skill allows the user to sidestep cyber-attacks by quickly rebooting all systems.

RESET

SHORT SKILL / ARO

[No LoF](/Labels "Labels")

REQUIREMENTS

Troopers can only Reset if at least one of these is true:

*   They are the Active Trooper.
*   In the Reactive Turn, if they are allowed to declare an [ARO](/ARO "ARO").

EFFECTS

*   Allows the user to make a [Face to Face Roll](/Face_to_Face_Roll "Face to Face Roll") to evade all enemy [Comms Attacks](/Comms_Attacks "Comms Attacks") and [BS Attacks (Guided)](/BS_Attack#BS_Attack_.28Guided.29 "BS Attack") during an [Order](/Order "Order") or [ARO](/ARO "ARO"), regardless of the [Burst](/Burst "Burst") (B) value.
*   This Face to Face Roll pits the user's [WIP](/WIP "WIP") Attribute against the attacker's WIP Attribute.
*   If the user is not making a Face to Face Roll (for example they are not the target of a Comms Attack), they will instead make a [Normal Roll](/Normal_Roll "Normal Roll") against their WIP Attribute.
*   Reset does not allow the user to evade other types of Attacks, but the [Dodge](/Dodge "Dodge") Skill does.
*   A successful Reset Roll, whether it's a Normal or Face to Face, allows the user to cancel their [Targeted State](/Targeted_State "Targeted State") and [IMM-B](/IMM-B "IMM-B") State, applying any State-specific MODs.

  

REMEMBER

A Reset only allows Face to Face Rolls against [Comms Attacks](/Comms_Attacks "Comms Attacks") and [BS Attacks (Guided)](/BS_Attack#BS_Attack_.28Guided.29 "BS Attack").
