The user of this Special Skill has received special training with a focus on silent movement, the art of remaining undetected, and the capacity to strike an unsuspecting enemy with unrelenting close combat attacks from out of the blue.

STEALTH

AUTOMATIC SKILL

[Optional.](/Labels "Labels")

REQUIREMENTS

*   The user of this Special Skill must be in his Active Turn.

EFFECTS

*   The user can declare [Cautious Movement](/Cautious_Movement "Cautious Movement") inside the [Zone of Control](/Zone_of_Control "Zone of Control") and [Hacking Areas](/Hacking_Area "Hacking Area") of enemy Models and Markers.
*   If the user declares Cautious Movement, [Idle](/Idle "Idle"), or a Basic Short Skill with the [Movement Label](/Movement_Label "Movement Label"), they only grant [AROs](/ARO "ARO") to enemies that have [LoF](/LoF "LoF") to them.
*   If any other Skill is declared, AROs are granted normally.
*   If the user enters [Silhouette contact](/Silhouette_contact "Silhouette contact") with an enemy without entering their LoF, and declares any of those other Skills, that enemy can only declare [CC Attack](/CC_Attack "CC Attack"), [Dodge](/Dodge "Dodge"), [Reset](/Reset "Reset"), or those Skills that can be used in [Engaged State](/Engaged_State "Engaged State").

  

REMEMBER

Stealth is not effective against Troopers with the [Combat Instinct](/Combat_Instinct "Combat Instinct") or [Sixth Sense](/Sixth_Sense "Sixth Sense") Special Skills, or against [Deployable Weapons](/Deployable_Weapon "Deployable Weapon") or pieces of Equipment.

  

SEE ALSO

**[Targeted State](/Targeted_State "Targeted State")**

*   **Restriction:** A Trooper in this state may not use the **Stealth** Special Skill.

  

FAQs
----

N5 FAQ

Version: 0.0.0, Oct 2025

**Q: Which Automatic Skills or Equipment have to be declared when in a Marker State?**

A: Automatic Skills and Equipment that alter how an Order activates a Trooper, or restrict AROs, must be declared when used in Marker State. For example using Stealth.

_Related Pages: [Camouflaged State](/Camouflaged_State "Camouflaged State"), [Impersonation State](/Impersonation_State "Impersonation State"), [Open and Private Information](/Open_and_Private_Information "Open and Private Information"), Stealth_

  

N5 FAQ

Version: 0.0.0, Oct 2025

**Q: If several Troopers are activated at once (for example a Coordinated Order or a Fireteam) and generate AROs, can any of the AROs target Active Troopers that are using Stealth?**

A: Yes, but the ARO will become an Idle if the Stealth user does not declare a Skill that allows AROs.
