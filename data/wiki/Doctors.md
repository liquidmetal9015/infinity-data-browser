Troopers with this Special Skill have the medical know-how to stabilize their unconscious comrades while in the field, saving their lives and returning them to active duty.

DOCTOR

SHORT SKILL

[Optional](/Labels "Labels")

REQUIREMENTS

*   Troopers using this Special Skill must be in [Silhouette contact](/Silhouette_contact "Silhouette contact") with their target.
*   The target must have the [Vitality](/Vitality "Vitality") Attribute and be in [Unconscious State](/Unconscious_State "Unconscious State").

EFFECTS

*   By passing a [Normal](/Normal "Normal") [WIP](/WIP "WIP") Roll, the target's Unconscious State is cancelled, by removing 1 single [Wound](/Wound "Wound") from those it has received, unless a rule or Skill specified otherwise. If the WIP Roll fails, the target automatically enters the [Dead State](/Dead_State "Dead State") and is removed from the game table.
*   Troopers may be recovered from the Unconscious State as many times as necessary, as long as the Doctor passes the **WIP** Roll.

**Update** _PDF 5.2, October 2025_  

*   A Trooper with the Doctor Special Skill can use it on themselves, to cancel any states that can be canceled by Doctor. However, this cannot be done if the Trooper is in a [Null](/Null "Null") State.
*   This Skill may present the following variations in the Unit Profile:
    *   **Doctor (2W):** Allows the target to remove 2 Wounds when their Unconscious State is cancelled.
    *   **Doctor (ReRoll WIP=X):** Once per Order, any failed Roll with this Skill may be rerolled, but applying the WIP Attribute indicated between brackets. The player must accept the result of this last Roll. This reroll does not provide a new ARO to the enemy.

    *   **Doctor (ReRoll \-X):** Once per Order, any failed Roll with this Skill may be rerolled, but with a negative MOD (-X) to the WIP Attribute. The player must accept the result of this last Roll. This reroll does not provide a new [ARO](/ARO "ARO") to the enemy.

SEE ALSO

Doctor can be used to cancel these States:

**[Unconscious](/Unconscious "Unconscious") State**

*   A [Wound](/Wound "Wound") is removed from the Trooper, leaving them with less Wounds than their [VITA](/VITA "VITA")/[STR](/STR "STR") Attribute value.
*   [Doctors](/Doctors "Doctors") can remove Wounds from Troopers with the VITA Attribute, while [Engineers](/Engineers "Engineers") can remove Wounds from Troopers with the STR Attribute. In both cases, the Wound is removed by performing the Doctor/Engineer Special Skill while in [Silhouette contact](/Silhouette_contact "Silhouette contact") with the affected Trooper, and passing a [Normal](/Normal "Normal") [WIP](/WIP "WIP") Roll (or the Roll specified by the Special Skill or Scenario).

  
**[Stunned](/Stunned "Stunned") State** (target does not need to be Unconscious)

*   To cancel this state, Troopers with the **[Vitality](/Vitality "Vitality") (VITA)** Attribute require a Doctor while Troopers with the **[Structure](/Structure "Structure") (STR)** Attribute require an [Engineer](/Engineer "Engineer").
*   A Trooper with the **Doctor/Engineer** Special Skill (or an equivalent Skill) may cancel this state by spending one [Short Skill](/Short_Skill "Short Skill") of an [Order](/Trooper_Activation "Trooper Activation") while in [Silhouette contact](/Silhouette_contact "Silhouette contact") with the affected Trooper and passing a [Normal](/Normal "Normal") [WIP](/WIP "WIP") Roll (or the Roll specified by the Special Skill or Scenario).

  

SEE ALSO

**[Command Tokens: Operational Use](/Command_Tokens#Command_Tokens:_Operational_Use "Command Tokens")** and **[Cubes](/Cube "Cube")**

If the target has a Cube, Command Tokens can be expended to reroll failed Doctor Rolls.

  
**[Technorganic](/Technorganic "Technorganic")**

*   The Unconscious State of the Trooper can be cancelled by using either the Doctor or [Engineer](/Engineer "Engineer") Skill, or the [MediKit](/MediKit "MediKit") or [GizmoKit](/GizmoKit "GizmoKit") pieces of Equipment, regardless of whether they have the [VITA](/VITA "VITA") or [STR](/STR "STR") Attribute.

  
**[Order Expenditure Sequence](/Order_Expenditure_Sequence "Order Expenditure Sequence")**

**Update** _PDF 5.2, October 2025_  

IMPORTANT

If a Trooper receives and removes [Wounds](/Wound "Wound") or [States](/States "States") in the same Order, the positive effect is always applied first, followed by the negative effect.

For example, if an [Engineer](/Engineer "Engineer") cancels a Trooper’s [IMM-A](/IMM-A "IMM-A") State at the same time that the same Trooper is hit by a Riotstopper, the Engineer’s WIP Roll to cancel the Trooper’s IMM-A State is made first, then the Trooper makes their PH-6 [Saving Roll](/Saving_Roll "Saving Roll") against the Riotstopper’s [PARA](/PARA "PARA") Special Ammunition. In this way, the original IMM-A Status could be canceled and a new one inflicted in the same Order.
