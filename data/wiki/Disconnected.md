DISCONNECTED

[Null](/Labels "Labels")

ACTIVATION

*   The Controller of this Trooper is in [Isolated State](/Isolated_State "Isolated State") or any [Null](/Null "Null") State.
*   The [Peripheral](/Peripheral "Peripheral") fails a [Coherency](/Coherency "Coherency") Check.
*   The Peripheral is subject to a successful Attack or Effect of an [Ammunition](/Ammunition "Ammunition"), [Hacking Program](/Hacking_Program "Hacking Program"), condition, or Scenario Special Rule that specifies that it causes this state.
*   The Peripheral is in [Isolated State](/Isolated_State "Isolated State").

EFFECTS

*   **Restriction:** The Peripheral cannot be activated or declare [Orders](/Trooper_Activation "Trooper Activation") or [AROs](/ARO "ARO").
*   Automatic Special Skills and Automatic Equipment have no effect while in this state, with the exception of the Peripheral Special Skill.
*   **Restriction:** The Peripheral does not count towards the Player's [Victory Points](/Victory_Points "Victory Points").

CANCELLATION

*   This state is automatically canceled if the Controller of this Peripheral recovers from the [Isolated State](/Isolated_State "Isolated State") or [Null](/Null "Null") State which activated the Disconnected State.
*   If this state was caused by an Attack, a piece of scenery, or a Scenario Special Rule applied to this Trooper, then a Trooper with the [Engineer](/Engineer "Engineer") Special Skill (or an equivalent Skill) may cancel this state by spending one Short Skill of an Order while in [Silhouette contact](/Silhouette_contact "Silhouette contact") with the affected Trooper, and passing a [Normal](/Normal "Normal") [WIP](/WIP "WIP") Roll (or the Roll specified by the Special Skill or Scenario).
*   If this state was caused by failing a [Coherency](/Coherency "Coherency") Check, it is automatically canceled by passing a Coherency Check at the end of an [Order](/Trooper_Activation "Trooper Activation") or [ARO](/ARO "ARO").
