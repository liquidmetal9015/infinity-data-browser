CYBERMASK

LONG SKILL

[NFB, No Roll.](/Labels "Labels")

REQUIREMENTS

*   The user must be outside the [LoF](/LoF "LoF") of enemy Markers or Troopers.

EFFECTS

*   The user enters the [Impersonation-2 state](/Impersonation_State "Impersonation State"), which does not require a Roll, replacing the user's model with an Impersonation-2 Marker (IMP-2).
*   The effects of this Program persist until the user reveals themselves as per the Impersonation-2 state rules.
*   This Program affects only the [Hacker](/Hacker "Hacker") himself and therefore has no range.

  

REMEMBER

While the [Hacker](/Hacker "Hacker") is in [IMP-2 State](/Impersonation_State "Impersonation State"), the Program’s [NFB](/NFB "NFB") Label applies, so they cannot use or apply any other Skill, piece of Equipment, or Hacking Program that has the NFB Label, for example [Mimetism](/Mimetism "Mimetism"), [Albedo](/Albedo "Albedo"), [Holoprojector](/Holoprojector "Holoprojector") etc.
