TRINITY

SHORT SKILLS / ARO

[Comms Attack](/Labels "Labels")

REQUIREMENTS

*   The target must be an enemy [Hacker](/Hacker "Hacker").

EFFECTS

*   In the Active Turn, this Program's [Burst](/Burst "Burst") of 3 allows the user to perform three [WIP](/WIP "WIP") Rolls, which can be distributed among one or more targets as usual.
*   The user applies a +3 WIP MOD to their Roll.
*   Each successful Roll forces the target to make a [Saving Roll](/Saving_Roll "Saving Roll") against [BTS](/BTS "BTS"), with PS 6.
*   For each failed Saving Roll, the target receives 1 [Wound](/Wound "Wound").
*   A [Critical](/Critical "Critical") with Trinity forces its target to perform an additional Saving Roll.
*   The range of this Program is the [Hacker](/Hacker "Hacker")'s [Hacking Area](/Hacking_Area "Hacking Area").
