Special piece of Equipment that allows other Models with the Structure Attribute to remove Wounds.

GIZMOKIT

SHORT SKILL

[Non-Lethal](/Traits "Traits")

REQUIREMENTS

*   The target of a GizmoKit must be an [Allied](/Allied "Allied") Model with the [STR](/STR "STR") Attribute.
*   A GizmoKit can be used in one of two ways, each one with their own Requirements:
    *   To use it remotely, the user must have LoF to the target.
    *   To use it as a piece of contact equipment, the user must be in Silhouette contact with the target.

EFFECTS

*   Used remotely, a GizmoKit is a Non-Lethal BS Weapon. If the user passes a [BS Attack](/BS_Attack "BS Attack") Roll, applying the corresponding MODs (for [Range](/Range "Range"), [Cover](/Cover "Cover"), etc.), the target performs a single [PH](/PH "PH") Roll.
*   A GizmoKit can also be used in [Silhouette contact](/Silhouette_contact "Silhouette contact"). To do so, the user must spend one Short Skill, without the need for a Roll, and their target performs a PH Roll.
*   In both cases, by passing the PH Roll, the target removes a [Wound](/Wound "Wound"). This cancels [Unconscious State](/Unconscious_State "Unconscious State") if applicable. If the PH Roll is failed, the target receives a Wound instead of removing it, entering Unconscious or [Dead State](/Dead_State "Dead State") if applicable.
*   The target of a GizmoKit does not perform a [Saving Roll](/Saving_Roll "Saving Roll").
*   A GizmoKit can be used as many times as necessary to remove all the Wounds that the target has.
*   If, due to a Special Skill or rule, a Trooper receives more than one successful hit or use of GizmoKit during the same Order, if any of the corresponding PH Rolls is successful, the Trooper removes a Wound. Even if the Trooper passes every PH Roll, they only remove one single Wound.

**Update** _PDF 5.2, October 2025_  

*   The user can apply this piece of Equipment on themselves, to remove any Wounds they have received, or cancel any states that can be canceled by a GizmoKit. However, this **cannot** be done if the user is in a **Null** State.

*   If a Trooper lists GizmoKit in their [Unit Profile](/Unit_Profile "Unit Profile") with a PH value between brackets, the Trooper will use this PH value when a GizmoKit is used on them.

### GizmoKit and Remote Presence

*   **GizmoKit and Remote Presence:**
    *   Regardless of which level of Unconscious State the target is in, with a single successful PH Roll, they remove all the Wounds necessary to cancel their Unconscious State.
