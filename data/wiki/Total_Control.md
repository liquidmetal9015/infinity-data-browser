TOTAL CONTROL

SHORT SKILL / ARO

[Comms Attack, Non-Lethal.](/Labels "Labels")

REQUIREMENTS

*   The target must be an [enemy](/Enemy "Enemy") [TAG](/TAG "TAG"), or a TAG in [Possessed State](/Possessed_State "Possessed State").

EFFECTS

*   **In the Active or Reactive Turn**, this Program has a [Burst](/Burst "Burst") of 1, allowing the user to perform one [WIP](/WIP "WIP") Roll against the target.
*   Each successful Roll, due to [DA Ammunition](/DA_Ammunition "DA Ammunition"), forces the target to make two [Saving Rolls](/Saving_Roll "Saving Roll") against [BTS](/BTS "BTS"), with PS 4.
*   Any failed Saving Roll causes the target to enter [Possessed State](/Possessed_State "Possessed State"), placing a Possessed State Token (POS) by them. If this Program is used against a TAG in Possessed State, any failed Saving Roll cancels the target's Possessed State, causing them to enter [Normal State](/Normal_State "Normal State"), and the Possessed State Token (POS) is removed.
*   A [Critical](/Critical "Critical") with Total Control forces its target to perform an additional Saving Roll.
*   The range of this Program is the [Hacker](/Hacker "Hacker")'s [Hacking Area](/Hacking_Area "Hacking Area").
