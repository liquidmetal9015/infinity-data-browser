FOXHOLE STATE

ACTIVATION

**Update** _PDF 5.2, October 2025_  

*   Automatic during the [Deployment Phase](/Deployment_Phase "Deployment Phase") for those Troopers with the [Sapper](/Sapper "Sapper") Special Skill.
*   During the Active Turn, Troopers may only activate this state by spending a [Long Skill](/Long_Skill "Long Skill") and placing a Foxhole Token beside them.

**Original**  

*   Automatic during the [Deployment Phase](/Deployment_Phase "Deployment Phase").
*   During the Active Turn, Troopers may only return to this state by spending a [Long Skill](/Long_Skill "Long Skill").

EFFECTS

*   The Trooper has a [Silhouette](/Silhouette "Silhouette") Attribute value of 3 (S3). If the Trooper's [Unit Profile](/Unit_Profile "Unit Profile") has a higher Silhouette Attribute value, they use that value while in this state.
*   The Trooper has [Partial Cover](/Partial_Cover "Partial Cover") in a 360˚ arc, from all directions and angles.
*   The Trooper has the [Mimetism](/Mimetism "Mimetism") (-3) and [Courage](/Courage "Courage") Special Skills.
*   Foxhole State is a fixed position which doesn't allow its user to perform any type of movement, including movement granted by a successful [Dodge](/Dodge "Dodge") Roll.

CANCELLATION

*   The Trooper enters [Prone State](/Prone_State "Prone State").
*   A Trooper in Foxhole State in the [Active Turn](/Active_Turn "Active Turn") may automatically cancel this state at the beginning of their movement by declaring a Skill with the [Movement Label](/Movement_Label "Movement Label"). The player must announce he is Canceling Foxhole State when declaring the Skill. By doing so, the state is canceled at no cost and the Trooper can carry out its Movement using its regular [MOV](/MOV "MOV") and [S](/S "S") values. In the same way, in the Active Turn, the Trooper may automatically cancel this state at the beginning of a [Dodge](/Dodge "Dodge") move.
*   When Canceling Foxhole State, the Trooper loses all the advantages provided by it, the Foxhole State Token is removed from the game table and the Trooper recovers his MOV and Silhouette values.

  

SEE ALSO

**[Sapper](/Sapper "Sapper")**

*   In the [Deployment Phase](/Deployment_Phase "Deployment Phase"), the user may deploy in Foxhole State, placing a Foxhole Token beside them.
*   In the [Active Turn](/Active_Turn "Active Turn"), a Trooper can use this [Long Skill](/Long_Skill "Long Skill") to enter Foxhole State and place a Foxhole Token beside them.

  

REMEMBER

The space in which the user of this Special Skill activates Foxhole State must have a height and width equal or greater than the [Silhouette Template](/Silhouette_Template "Silhouette Template") of this state.
