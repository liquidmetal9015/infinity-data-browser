Engineers have the means and the skill to repair damaged Equipment on the field.

ENGINEER

SHORT SKILL

[Optional](/Labels "Labels")

REQUIREMENTS

*   Troopers using this Special Skill must be in [Silhouette contact](/Silhouette_contact "Silhouette contact") with their target.

EFFECTS

*   Engineer allows, by passing a [Normal](/Normal "Normal") [WIP](/WIP "WIP") Roll, removing a [Wound](/Wound "Wound") from a target with the [Structure](/Structure "Structure") Attribute (STR). This cancels [Unconscious State](/Unconscious_State "Unconscious State") if applicable.
*   This Special Skill can be declared as many times as necessary to remove all the Wounds that the target has.
*   If the Engineer fails the Normal WIP Roll, the target receives a Wound instead of removing it, entering Unconscious or [Dead State](/Dead_State "Dead State") if applicable.
*   Alternatively, by passing a Normal WIP Roll, a Trooper with Engineer may cancel all of their target's [States](/States "States") that can be canceled by this Special Skill except Unconscious ([Immobilized](/Immobilized "Immobilized")\-A and B, [Targeted](/Targeted "Targeted"), etc.), removing the corresponding State Tokens.
*   Failing the Normal WIP Roll when trying to cancel other states has no negative consequence, and the Roll can be performed again in future Orders.

**Update** _PDF 5.2, October 2025_  

*   A Trooper with the Engineer Special Skill can use it on themselves, to remove any Wounds they have received, or cancel any states that can be canceled by Engineer. However, this cannot be done if the Trooper is in a [Null](/Null "Null") State.
*   **Engineer (ReRoll WIP=X):**  
    Once per Order, any failed Roll with this Skill may be rerolled, but applying the WIP Attribute indicated between brackets. The player must accept the result of this last Roll. This reroll does not provide a new ARO to the enemy.

**Original**  

*   **Engineer (ReRoll \-X):**  
    Once per Order, any failed Roll with this Skill may be rerolled, but with a negative MOD (-X) to the WIP Attribute. The player must accept the result of this last Roll. This reroll does not provide a new ARO to the enemy.

*   **Engineer and [Remote Presence](/Remote_Presence "Remote Presence"):**  
    Regardless of which level of Unconscious State the target is in, with a single successful WIP Roll, they remove all the Wounds necessary to cancel the target's Unconscious State.

However, the Attribute value provided by the Engineer (ReRoll WIP=X) Skill does not apply when repeating this Roll.

However, the negative MOD of the Engineer (ReRoll -X) Skill does not apply when repeating this Roll.

SEE ALSO

**[Command Tokens: Operational Use](/Command_Tokens#Command_Tokens:_Operational_Use "Command Tokens")** and **[Remote Presence](/Remote_Presence "Remote Presence")**

If the target has Remote Presence, Command Tokens can be expended to reroll failed Engineer Rolls.

  
**[Technorganic](/Technorganic "Technorganic")**

*   The Unconscious State of the Trooper can be cancelled by using either the [Doctor](/Doctor "Doctor") or Engineer Skill, or the [MediKit](/MediKit "MediKit") or [GizmoKit](/GizmoKit "GizmoKit") pieces of Equipment, regardless of whether they have the [VITA](/VITA "VITA") or [STR](/STR "STR") Attribute.

  
**[Order Expenditure Sequence](/Order_Expenditure_Sequence "Order Expenditure Sequence")**

**Update** _PDF 5.2, October 2025_  

IMPORTANT

If a Trooper receives and removes [Wounds](/Wound "Wound") or [States](/States "States") in the same Order, the positive effect is always applied first, followed by the negative effect.

For example, if an Engineer cancels a Trooper’s [IMM-A](/IMM-A "IMM-A") State at the same time that the same Trooper is hit by a Riotstopper, the Engineer’s WIP Roll to cancel the Trooper’s IMM-A State is made first, then the Trooper makes their PH-6 [Saving Roll](/Saving_Roll "Saving Roll") against the Riotstopper’s [PARA](/PARA "PARA") Special Ammunition. In this way, the original IMM-A Status could be canceled and a new one inflicted in the same Order.
