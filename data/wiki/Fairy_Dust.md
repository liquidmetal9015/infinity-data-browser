FAIRY DUST

LONG SKILL

[Supportware, No Roll.](/Labels "Labels")

REQUIREMENTS

*   Only [HIs](/HI "HI"), [REMs](/REM "REM"), [TAGs](/TAG "TAG"), or [VHs](/VH "VH") from the same [Army List](/Army_List "Army List") as the user may be chosen as targets.

EFFECTS

*   Fairy Dust allows the Player to choose a single [Troop Type](/Troop_Type "Troop Type") (HI, REM, TAG, or VH) that will benefit from **[Firewall](/Firewall "Firewall") (-3)**. Every Trooper of the [Hacker](/Hacker "Hacker")'s Army List that belongs to the chosen Troop Type applies the corresponding MODs every time they suffer a [Comms Attack](/Comms_Attack "Comms Attack").
*   When the user activates this Program, which does not require a Roll, place a Fairy Dust Token (SUP: FAIRYDUST) beside them.
*   This Program's range covers the entire game table.

IMPORTANT

**SUPPORTWARE**

*   A Trooper can benefit from the effects of only one [Program](/Hacking_Program "Hacking Program") with the [Supportware Label](/Supportware_Label "Supportware Label") at a time.
*   Each [Hacker](/Hacker "Hacker") can only sustain one active Supportware Program at a time.
*   A Hacker can cancel an active Supportware Program by declaring another Program and expending the corresponding [Order](/Trooper_Activation "Trooper Activation") or [ARO](/ARO "ARO").
*   A Supportware Program is automatically cancelled if the target of the Supportware is targeted by a new Supportware Program, or if the Hacker running the Program enters [Isolated State](/Isolated_State "Isolated State") or any [Null](/Null "Null") States.

  

Example of Supportware Use and Cancellation

During their Active Turn, a Player that has two Troopers with [EVO Hacking Devices](/EVO_Hacking_Device "EVO Hacking Device") on the table, declares Fairy Dust with one of them and chooses [REMs](/REM "REM") as the [Troop Type](/Troop_Type "Troop Type") that will benefit from this Supportware.

For tactical reasons, they decide to spend the last Order on the other Trooper with an EVO Hacking Device to declare [Enhanced Reaction](/Enhanced_Reaction "Enhanced Reaction") on a REM located in the vanguard.

By designating a REM that was already benefitting from Fairy Dust, the new Supportware Program deactivates the Fairy Dust Program and the corresponding Token is removed from the table.

  

REMEMBER

A Supportware Program is automatically cancelled if the target of the Supportware is targeted by a new Supportware Program, or if the [Hacker](/Hacker "Hacker") running the Program enters [Isolated](/Isolated "Isolated") State or any [Null](/Null "Null") States.
