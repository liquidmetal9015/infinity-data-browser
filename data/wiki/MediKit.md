**Original**  
Special piece of Equipment that allows other Models to regain consciousness.

MEDIKIT

SHORT SKILL

[Non-Lethal](/Traits "Traits")

REQUIREMENTS

*   A MediKit's target must be an [Allied](/Allied "Allied") Model with the [VITA](/VITA "VITA") Attribute and in [Unconscious State](/Unconscious_State "Unconscious State"), unless certain scenario rules say otherwise.
*   A MediKit can be used in one of two ways, each one with their own Requirements:
    *   To use it remotely, the user must have [LoF](/LoF "LoF") to the target.
    *   To use it as a piece of contact equipment, the user must be in [Silhouette contact](/Silhouette_contact "Silhouette contact") with the target.

EFFECTS

*   Used remotely, a MediKit is a Non-Lethal BS Weapon. If the user passes a [BS Attack](/BS_Attack "BS Attack") Roll, applying the corresponding MODs (for [Range](/Range "Range"), [Cover](/Cover "Cover"), etc.), the target performs a single [PH](/PH "PH") Roll.
*   A MediKit can also be used in [Silhouette contact](/Silhouette_contact "Silhouette contact"). To do so, the user must spend one Short Skill without needing a Roll, and their target performs a PH Roll.
*   In both cases, by passing the PH Roll, the target may remove one [Wound](/Wound "Wound") and cancel their [Unconscious State](/Unconscious_State "Unconscious State"). If the PH Roll is failed, the target automatically enters [Dead State](/Dead_State "Dead State") and is removed from the game table.
*   The target of a MediKit does not perform a [Saving Roll](/Saving_Roll "Saving Roll").
*   A MediKit can be used as many times as necessary to cancel a Trooper's Unconscious State, as long as they pass the corresponding PH Roll.
*   If, due to a Special Skill or rule, a Trooper receives more than one successful hit or use of MediKit during the same Order, if any of the corresponding PH Rolls is successful, the Trooper removes a Wound and cancels their Unconscious State. In this case, even if the Trooper passes every PH Roll, they only remove one single Wound.

**Update** _PDF 5.2, October 2025_  

*   The user can apply the MediKit on themselves, to cancel any states that can be canceled by this piece of Equipment. However, this **cannot** be done if the user is in a **[Null](/Null "Null")** State.
*   If a Trooper lists MediKit in their [Unit Profile](/Unit_Profile "Unit Profile") with a [PH](/PH "PH") value between brackets, the Trooper will use this PH value when a MediKit is used on them.
