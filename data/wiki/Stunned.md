STUNNED STATE

ACTIVATION

*   The Trooper or game element ("Trooper" from now on) suffers a successful [Attack](/Attack "Attack") or Effect using an [Ammunition](/Ammunition "Ammunition"), [Hacking Program](/Hacking_Program "Hacking Program"), or a game condition or Scenario Special Rule, capable of causing this state.

EFFECTS

*   **Restriction:** Troopers in this state cannot declare [Attacks](/Attack "Attack").
*   Moreover, Troopers in Stunned State must apply a MOD of -3 to any Roll they perform, except [Saving Rolls](/Saving_Rolls "Saving Rolls").
*   Automatic Special Skills and Equipment continue to work, but the Trooper must still respect all declaration restrictions.

CANCELLATION

*   To cancel this state, Troopers with the **[Vitality](/Vitality "Vitality") (VITA)** Attribute require a [Doctor](/Doctor "Doctor") while Troopers with the **[Structure](/Structure "Structure") (STR)** Attribute require an [Engineer](/Engineer "Engineer").
*   A Trooper with the **Doctor/Engineer** Special Skill (or an equivalent Skill) may cancel this state by spending one [Short Skill](/Short_Skill "Short Skill") of an [Order](/Trooper_Activation "Trooper Activation") while in [Silhouette contact](/Silhouette_contact "Silhouette contact") with the affected Trooper and passing a [Normal](/Normal "Normal") [WIP](/WIP "WIP") Roll (or the Roll specified by the Special Skill or Scenario).
*   The Stunned State is automatically canceled at the start of the [States Phase](/States_Phase "States Phase") of the [Player Turn](/Player_Turn "Player Turn") in which it was caused.

  

SEE ALSO

**[Stun Ammunition](/Stun_Ammunition "Stun Ammunition")**

*   A failed Saving Roll against Stun Ammunition causes the target to enter Stunned State. Place a Stunned State Token (STUNNED) next to it.

  
**[Cybermines](/Mines#Cybermines "Mines")**

*   Troopers who are hit by a Cybermine must make **two** [Saving Rolls](/Saving_Rolls "Saving Rolls") against [BTS](/BTS "BTS"), with PS 5. If the target fails either [Saving Roll](/Saving_Roll "Saving Roll"), it enters **[Stunned](/Stunned "Stunned")** State.
*   However, any Troopers with the **[Hackable](/Hackable "Hackable") Characteristic ([HI](/HI "HI"), [REM](/REM "REM"), [TAG](/TAG "TAG"), [VH](/VH "VH")...)** and **[Hackers](/Hackers "Hackers")** who fail either Saving Roll enter **[Immobilized-B](/Immobilized-B "Immobilized-B")** State instead of Stunned State.
