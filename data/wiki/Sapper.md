This Special Skill allows its user to deploy in a fortified emplacement that provides some in-game advantages.

SAPPER

DEPLOYMENT SKILL / LONG SKILL

[Optional](/Labels "Labels")

REQUIREMENTS

*   The space in which the user activates [Foxhole State](/Foxhole_State "Foxhole State") must have a height and width equal or greater than the [Silhouette Template](/Silhouette_Template "Silhouette Template") of this State.
*   Otherwise, the Trooper will not be able to enter Foxhole State and will perform an [Idle](/Idle "Idle") instead.

EFFECTS

*   In the [Deployment Phase](/Deployment_Phase "Deployment Phase"), the user may deploy in Foxhole State, placing a Foxhole Token beside them.
*   In the [Active Turn](/Active_Turn "Active Turn"), a Trooper can use this [Long Skill](/Long_Skill "Long Skill") to enter Foxhole State and place a Foxhole Token beside them.
